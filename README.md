# geometry-tolerance-calculator
#### 功能描述

基于三坐标测量实验计算几何公差的代码。

学校开设的三坐标几何公差测量实验要求代码实现最小二乘法计算相应的几何公差。顺手写了个函数。实现多种几何公差的计算。

#### 使用流程

- 确保已经安装python
  - python的安装我就不多插嘴了，网上搜索，教程一大堆
- 获取代码 git clone
- 更改 calculator.py 中相关参数
  - 你的文件名称
    - 注意，如不是在同一文件目录下，请搬迁到同一目录，或者使用绝对路径
  - 你的计算模式
- run
